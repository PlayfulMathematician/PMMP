## Table of Contents
 - [Installation](#Installation)
 - [Docs](#Docs)


# Installation 
## MacOS
```bash
$ pip3.11 install PMMP
```
## Windows
```bash
$ pip install PMMP
```
## Linux
```bash
$ pip3 install PMMP
```
